# GDG
GDG (Grafana Dash-n-Grab) 
est l'outil moderne, écrit en Go, qui gère parfaitement 
les nouvelles API de Grafana (Service Accounts, UID stables,
dossiers imbriqués). C'est le couteau suisse parfait pour 
votre workflow sur AlmaLinux.

```shell
wget https://github.com/esnet/gdg/releases/download/v0.9.2/gdg_Linux_x86_64.tar.gz
tar -xvf gdg_Linux_x86_64.tar.gz
chmod +x gdg
sudo mv gdg /usr/local/bin/gdg
gdg version
```
